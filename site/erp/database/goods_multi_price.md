## 商品多价格信息同步
### 1.获取商品对应门店库存信息
#### 1.1 结构描述
> 定时批量读取门店商品价格，用于更新系统方线上商品门店价格，暂定按2小时同步一次
#### 1.2 请求方式
> sql-select
#### 1.3 url
> 商户方数据库地址，端口，数据库名称，访问用户，访问密码
#### 1.4 数据方向
> 商户方至系统方
#### 1.5 查询结构
| 参数名称 | 参数类型 | 是否必须 | 示例值 | 参数描述  |
| :---         |     :---      |     :--- | :--- | :--- |
| goodsno   | 字符串     | 是    | 90301104    | 商品编码 |
| uid   | 字符串    | 是    | 2    | 门店编号 |
| goodprice   | 字符串    | 是    | 33.8    | 门店商品价格 |
--------------------- 
#### 1.5 返回字段
| 参数名称 | 参数类型 | 是否必须 | 示例值 | 参数描述  |
| :---         |     :---      |     :--- | :--- | :--- |
| goodsno   | 字符串     | 是    | 90301104    | 商品编码 |
| uid   | 字符串    | 是    | 2    | 门店编号 |
| goodprice   | 字符串    | 是    | 33.8    | 门店商品价格 |
---------------------
