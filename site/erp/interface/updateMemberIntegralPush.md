## 主动推送会员积分接口



1.**会员积分变动**

#### 1.1 接口描述

- erp会员变动,主动推送到线上系统
- accessKey在商户后台手动生成，然后去申请accessToken

1.2 请求方式

> post

#### 1.3 url

> /member/updateMemberIntegralPush

#### 1.4 数据方向

> **erp**-->线上系统

#### 1.5 请求参数

| 参数名称    | 参数类型 | 是否必须 | 示例值               | 参数描述        |
| :---------- | :------- | :------- | :------------------- | --------------- |
| accessToken | 字符串   | 是       | acfde325c8b9bbgyR57B | 由accessKey生成 |
| siteId      | 字符串   | 是       | 100190               | 商户ID          |
| mobile      | 字符串   | 是       | 13899999999          | 手机号          |
| integrate   | 字符串   | 是       | 100                  | 积分            |

#### 1.6 请求示例

```java
{
    "siteId":"100190",
    "accessToken":"acfde325c8b9bbgyR57B",
     "integrateList":[{
       "mobile":"13899999999",
       "integrate":"100"
     },
     {
      "mobile":"13689999999",
       "integrate"："100"
     }]
}

```

------



#### 1.7 返回示例(请求成功)

```
{
  "code":"10000",
  "msg": "推送成功"
}
```

#### 1.8 返回示例(请求失败)

```
{
  "code":"20000",
  "msg": "推送失败,相关原因！"
}
```
